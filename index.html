Atividade de Web:

Conheça algumas funções

Web nome pelo qual a rede mundial de computadores internet se tornou conhecida a partir de 1991, quando se popularizou devido à criação de uma interface gráfica que facilitou o acesso e estendeu seu alcance ao público em geral.

1. Como funciona a regra da cascata no CSS?

    A cascata define como o navegador escolhe qual estilo aplicar quando existem várias regras para o mesmo elemento. A ordem de prioridade é:

  1. Importância → Regras !important têm prioridade máxima.

  2. Especificidade → Regras mais específicas substituem as mais genéricas.

  3. Origem → Estilos inline (dentro do HTML) têm prioridade sobre os internos style e externos arquivos CSS.

  4. Ordem de aparição → Se duas regras tiverem a mesma especificidade, a última lida pelo navegador prevalece.

2. O que é especificidade

Especificidade é um sistema de pontos usado para determinar qual regra CSS tem mais “peso”.

  • Seletor por tag (h1, p, div…) → especificidade baixa.

  • Seletor por classe (.nome), pseudo-classes (:hover, :first-child…) → especificidade média.

  • Seletor por id (#id) → especificidade alta.
Exemplo: h1 { color: blue; } /* peso baixo */ .titulo { color: green; } /* peso médio */ #principal { color: red; } /* peso alto */ O #principal vence.

3. Combinação de seletores

  • Descendente (espaço) → .primeiro h1 Aplica estilo a qualquer h1> dentro de um elemento com classe .primeiro.

  • primeiro h1 { color: blue; }

  • Filho direto ( > ) → .primeira > h1 Aplica estilo apenas ao h1> que é filho imediato da primeira.

  • primeira > h1 { color: red; }

  • Irmão adjacente ( + ) → .primeira h1 + h1 Aplica estilo ao h1> que vem logo depois de outro h1> dentro de .primeira.

  • primeira h1 + h1 { color: green; }

4. Pseudo-classes

  1. :first-child → seleciona o primeiro filho de um elemento.
  2. :last-child → seleciona o último filho.
  3. :nth-child(n) → seleciona o filho na posição n. (Ex: :nth-child(2) = segundo filho).
  4. :nth-of-type(n) → parecido, mas conta apenas filhos do mesmo tipo de elemento.

5. Diferença entre display inline e display block

    inline → ocupa apenas o espaço do conteúdo, não quebra linha. Não aceita width, height, margin-top, margin-bottom.
    block → ocupa toda a largura disponível e sempre quebra a linha. Aceita todas as propriedades de tamanho e espaçamento.

  • Exemplo:
  1. span { display: inline; } /* padrão */
  2. div { display: block; } /* padrão */

6. Elementos padrão inline e block

• Inline: span>, a>, img>, strong>, em>, label>.
• Block: div>, p>, h1>-h6>, section>, article>, header>, footer>.

7. Margin, Padding e Border

• margin → espaço externo ao redor do elemento.
• padding → espaço interno entre o conteúdo e a borda.
• border → a borda do elemento em si.

Shorthand (atalhos):
• 1 valor → aplica a todos os lados.
margin: 10px; /* 10px em todos os lados */

• 2 valores → primeiro valor = vertical (top/bottom), segundo = horizontal (left/right).
margin: 10px 20px;

• 3 valores → topo / laterais / baixo.
margin:: 10px 20px 30px;

• 4 valores → topo / direita / baixo / esquerda (sentido horário).
margin: margin: 10px 20px 30px 40px;

Sufixos: • -top, -right, -bottom, -left → aplicam a um lado específico.
• -inline, -block → controlam espaçamento em eixos inline (horizontal em idiomas LTR) e block (vertical).

8. box-sizing: border-box

Por padrão (content-box), o width e height aplicam apenas ao conteúdo, somando padding e border ao tamanho final.
Com box-sizing: border-box, o navegador inclui padding e border dentro do width/height → facilita muito no layout.

Exemplo:
div { width: 200px;
padding: 20px;
border: 5px solid black;
box-sizing: border-box;
} Nesse caso, o elemento terá 200px no total, e não 200 + 20 + 20 + 5 + 5.

Parque Yongdusan

O Parque Yongdusan, localizado no centro de Busan, abriga alguns dos monumentos mais importantes da cidade. Você pode ver vistas espetaculares do topo da Torre Busan, de 120 metros de altura. O parque tem 2 museus – confira os instrumentos musicais tradicionais no Museum of World Folk Instruments e mais de 80 veleiros coreanos no Exhibition Hall of World Model Boats.

Bom para:

  • Casais
  • Famílias
  • Orçamento

As melhores coisas para fazer em Busan mostram a reputação da cidade como um importante porto marítimo na Ásia. Frequentemente vista como a essência da Coreia do Sul, você experimentará uma atmosfera única em termos de diversidade étnica e cultural, já que a cidade recebe um público cosmopolita o ano todo.
