1. Como funciona a regra da cascata no CSS?

A cascata define como o navegador escolhe qual estilo aplicar quando existem várias regras para o mesmo elemento. A ordem de prioridade é:
1. Importância → Regras !important têm prioridade máxima.
2. Especificidade → Regras mais específicas substituem as mais genéricas.
3. Origem → Estilos inline (dentro do HTML) têm prioridade sobre os internos style e externos arquivos CSS.
4. Ordem de aparição → Se duas regras tiverem a mesma especificidade, a última lida pelo navegador prevalece.

2. O que é especificidade

Especificidade é um sistema de pontos usado para determinar qual regra CSS tem mais “peso”.
• Seletor por tag (h1, p, div…) → especificidade baixa.
• Seletor por classe (.nome), pseudo-classes (:hover, :first-child…) → especificidade média.
• Seletor por id (#id) → especificidade alta. Exemplo: h1 { color: blue; } /* peso baixo */ .titulo { color: green; } /* peso médio */ #principal { color: red; } /* peso alto */ O #principal vence.

3. Combinação de seletores

• Descendente (espaço) → .primeiro h1 Aplica estilo a qualquer

dentro de um elemento com classe .primeiro. .primeiro h1 { color: blue; } • Filho direto ( > ) → .primeira > h1 Aplica estilo apenas ao

que é filho imediato de .primeira. .primeira > h1 { color: red; } • Irmão adjacente ( + ) → .primeira h1 + h1 Aplica estilo ao

que vem logo depois de outro

dentro de .primeira. .primeira h1 + h1 { color: green; }

Parque Yongdusan

O Parque Yongdusan, localizado no centro de Busan, abriga alguns dos monumentos mais importantes da cidade. Você pode ver vistas espetaculares do topo da Torre Busan, de 120 metros de altura. O parque tem 2 museus – confira os instrumentos musicais tradicionais no Museum of World Folk Instruments e mais de 80 veleiros coreanos no Exhibition Hall of World Model Boats.

Bom para:

  • Famílias
  • Orçamento

As melhores coisas para fazer em Busan mostram a reputação da cidade como um importante porto marítimo na Ásia. Frequentemente vista como a essência da Coreia do Sul, você experimentará uma atmosfera única em termos de diversidade étnica e cultural, já que a cidade recebe um público cosmopolita o ano todo.
